### What is not a KPI

The following are tracked as context signals but carry no decision weight in this framework:

- Social follower count alone: tracked in monthly reports; never decision-triggering by itself
- Total website page impressions without a click or downstream conversion
- Bounce rate: useful as a diagnostic input during a specific investigation, not as a standing monthly metric
- Time on page: no actionable decision available at this school's current stage
- Newsletter open rate alone, without a linked inquiry or downstream action
- General Instagram impressions without an associated bio-link click, DM, or tour inquiry

## 5. Reporting cadence

### Weekly (Zach, internal)

Every Monday during the summer build. Shifts to every other Monday once all Phase 1 and Phase 2 automations have been stable for 30 consecutive days.

Duration: 30 minutes. Focus:
- automation_logs review: errors, escalations, volume per automation
- Claude API usage and cost vs. provisioned cap
- Any items requiring Rebecca's attention before the next business day
- Flag anything that crossed a Section 3 action threshold

Output: Internal Zach note filed to Joshua. Any threshold-crossing item is emailed to Rebecca the same day with a plain-English description of the issue and the expected resolution.

### Monthly (Joshua to Rebecca, aligned to the board report)

First week of every month. Duration to prepare: 2-3 hours.

The delivery timing aligns to Rebecca's monthly board report so she can incorporate the summary without requiring a separate communication cycle.

Report sections:
1. **One-paragraph executive summary:** What happened this month, what worked, what did not.
2. **KPI snapshot table:** Baseline vs. current vs. target for every active metric.
3. **Funnel report:** Inquiry count, tours scheduled, enrollments, trend vs. prior month and vs. baseline.
4. **Automation health:** Success rates per automation, escalation volume, notable issues, Claude API cost.
5. **SEO and content:** Keyword position changes, new indexed pages, traffic trend (once analytics is live).
6. **Social summary:** Post cadence vs. plan, engagement trend, DM inquiries sourced to Facebook and Instagram.
7. **Reviews:** New review count, average rating, response rate.
8. **Threshold flags:** Any metric that crossed a Section 3 action threshold this month, with the decision that was made.
9. **Recommended actions for next 30 days:** Prioritized list, no more than 5 items.

Delivery format: PDF emailed to Rebecca. The Google Sheet dashboard is linked for reference. An optional 30-minute review call is available; I will not schedule one by default unless there are threshold flags to discuss.

### Quarterly strategic review (Joshua and Rebecca)

Every 3 months. 90-minute meeting. Focus:

1. Three-month lookback: every KPI plotted against baseline.
2. What worked and what did not (an opinionated assessment, not just a data summary).
3. Source document refresh needs: which of Docs 01-11 require updating given changes in the school year, staffing, enrollment, or program offerings.
4. Competitor landscape changes: review whether Doc 05 needs a refresh.
5. Automation upgrade candidates: which Phase 3 or Phase 4 automations are ready to build (consent form database readiness for Photo Workflow, compliance data source status for Automation 14, etc.).
6. Strategic priorities for next quarter: 3 to 5 focus areas.
7. Upriver retainer scope: whether the light retainer through the fall needs to extend, reduce, or transition to a maintenance mode.

Output: Updated Doc 12 with revised targets; any source docs refreshed as needed per their individual schedules.

That is when I will set the first absolute targets for inquiry-to-enrollment conversion to replace the relative-threshold placeholders in Section 2.

### Escalation paths

- **Technical (automation failures, API errors, n8n issues):** Zach, within 24 hours. Zach escalates to Joshua if a prompt or logic revision is required.
- **Content (brand voice drift, factual error in auto-responder output, prohibited content flagged in any output):** Joshua, within 48 hours of the flag.
